# 📊 Data Profiler Project (EDA + YData Profiling)

## 🚀 Overview

This project focuses on **Exploratory Data Analysis (EDA)** and **automated data profiling** using Python. It analyzes datasets from multiple sources including CSV, JSON, and SQLite databases.

---

## 📂 Project Structure

* **CSV Dataset** → Player-level match data
* **JSON Dataset** → Match-level data
* **SQLite Database** → Structured relational data

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SQLite3
* YData Profiling

---

## 📊 Features

* Data Cleaning & Preprocessing
* Handling Missing Values
* Statistical Analysis
* Data Visualization (Histogram, Scatter Plot, etc.)
* Correlation Analysis
* Automated Profiling Report

---

## 📈 Key Insights

* Player performance analyzed using runs, strike rate, and wickets
* Distribution of runs and match outcomes explored
* Correlation between numerical features identified
* Dataset quality assessed using profiling tools

---

## 📄 Output Reports

* `csv_data_profile_report.html` → Full automated EDA report
* (Optional) JSON profiling report

---

## ▶️ How to Run

1. Install dependencies:

```bash
pip install pandas matplotlib seaborn ydata-profiling
```

2. Run the Jupyter Notebook:

```bash
jupyter notebook
```

3. Open:

```
Data_Profiler_Project.ipynb
```

## 💡 Conclusion

This project demonstrates how to perform complete EDA and generate professional data profiling reports using Python, making it useful for real-world data analysis tasks.

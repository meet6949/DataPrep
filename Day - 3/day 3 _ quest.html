What is Exploratory Data Analysis?
+30 XP on scroll
Core Definition
EDA (Exploratory Data Analysis) = data ko samajhne ki process — model banane se pehle patterns, anomalies, distributions aur relationships dhundhna.

Real analogy: Doctor pehle patient ka full checkup karta hai — blood test, BP, temperature — phir diagnosis deta hai. EDA bhi waisa hi hai — pehle data ka checkup, phir ML model!
The Golden Rule of Data Science
"Never trust your data blindly!" — EDA is that critical first step which saves hours of debugging later.
Data Scientists apna 60–70% time EDA mein lagate hain. Bina EDA ke model banana = andha maarna!
EDA ke 5 Main Goals
📐 Data ka Shape aur Structure samjho
Kitni rows? Kitne columns? Kaunse datatypes hain? Koi missing values toh nahi?
Tools: df.shape, df.info(), df.dtypes, df.describe()
📊 Distribution samjho
Kya data normally distributed hai? Skewed hai? Outliers hain?
Tools: Histograms, Boxplots, df['col'].skew()
🔗 Relationships dhundho
Kaunse features ek dusre se connected hain? Price aur Area ka kya relation?
Tools: Scatter plots, Heatmaps, Correlation matrix
🚨 Anomalies aur Outliers pakdo
Ek ₹50 crore ka flat average distort kar raha hai? Age -5 hai kisi ka?
Tools: Boxplots, IQR method, Z-score
💡 Hypotheses generate karo
"Lagta hai ki mumbai mein prices higher hain" — ye EDA se milta hai, phir model confirm karta hai!
EDA = Question Generator. Model = Answer Machine.

Univariate — Plots aur Python Code
Numerical — Histogram & KDE Plot
🐍 Python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv('ipl_2023.csv')

fig, axes = plt.subplots(1, 3, figsize=(15, 5))

# ─── Histogram ───
axes[0].hist(df['runs_scored'], bins=20, color='#2563eb', edgecolor='white', alpha=0.85)
axes[0].set_title('Runs Scored — Histogram')
axes[0].set_xlabel('Runs'); axes[0].set_ylabel('Frequency')

# ─── KDE Plot (Smooth distribution) ───
sns.kdeplot(df['runs_scored'], ax=axes[1], color='#7c3aed', fill=True, alpha=0.4)
axes[1].set_title('Runs Scored — KDE (Smooth Curve)')

# ─── Box Plot (Outliers!) ───
axes[2].boxplot(df['runs_scored'], patch_artist=True,
                boxprops=dict(facecolor='#cffafe', color='#0891b2'),
                medianprops=dict(color='#db2777', linewidth=2))
axes[2].set_title('Runs Scored — Box Plot')

plt.tight_layout()
plt.show()

# ─── Numerical Stats ───
print(df['runs_scored'].describe())
print(f"Skewness: {df['runs_scored'].skew():.2f}")
print(f"Kurtosis: {df['runs_scored'].kurtosis():.2f}")
📊 Box plot mein: Box = IQR (25th–75th percentile) | Line = Median | Dots = Outliers! Ek plot mein poori story!
Categorical — Bar Chart & Value Counts
🐍 Python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv('ecommerce_orders.csv')

# ─── Value Counts ───
print(df['product_category'].value_counts())
print(df['product_category'].value_counts(normalize=True) * 100)  # %

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# ─── Bar Chart ───
category_counts = df['product_category'].value_counts()
axes[0].bar(category_counts.index, category_counts.values,
            color=sns.color_palette("Set2", len(category_counts)))
axes[0].set_title('Orders by Category')
axes[0].set_xlabel('Category'); axes[0].set_ylabel('Count')
axes[0].tick_params(axis='x', rotation=45)

# ─── Seaborn Countplot (even easier!) ───
sns.countplot(data=df, x='product_category',
              palette='Set2', order=df['product_category'].value_counts().index,
              ax=axes[1])
axes[1].set_title('Seaborn Countplot — Same Data, Less Code!')
axes[1].tick_params(axis='x', rotation=45)

plt.tight_layout()
plt.show()
💡 value_counts(normalize=True) se percentage milti hai — "Electronics = 35% orders" — business ke liye direct insight!

Correlation — Relationship ki Strength
Correlation coefficient r ranges from -1 to +1:

r = +1 Perfect positive — ek badhta hai toh doosra bhi
r = 0 No relation — dono independent hain
r = -1 Perfect negative — ek badhta hai toh doosra ghatta hai

🏠 Mumbai: Area aur Price ka r = +0.82 → Strong positive correlation!

Numerical vs Numerical — Scatter Plot & Correlation
🐍 Python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv('mumbai_housing.csv')

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# ─── Scatter Plot ───
axes[0].scatter(df['area_sqft'], df['price_lakhs'],
                alpha=0.5, color='#2563eb', edgecolors='white', linewidth=0.5)
axes[0].set_xlabel('Area (sq ft)')
axes[0].set_ylabel('Price (₹ Lakhs)')
axes[0].set_title('Area vs Price — Mumbai Flats')

# ─── Seaborn regplot (scatter + regression line) ───
sns.regplot(data=df, x='area_sqft', y='price_lakhs',
            ax=axes[1], color='#7c3aed',
            scatter_kws={'alpha': 0.4},
            line_kws={'color': '#db2777', 'linewidth': 2})
axes[1].set_title('Area vs Price — With Trend Line')

plt.tight_layout(); plt.show()

# ─── Correlation value ───
corr = df['area_sqft'].corr(df['price_lakhs'])
print(f"Pearson Correlation: {corr:.3f}")   # 0.82 → Strong positive!
📈 regplot = scatter + linear trend line ek saath! Positive slope = positive correlation. Real Mumbai data mein r ≈ 0.75!

import pandas as pd

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

plt.tight_layout(); plt.show()

# ─── Mean per group ───
print(df.groupby('team')['runs_scored'].mean().sort_values(ascending=False))

Multivariate Analysis
+30 XP on scroll
Multivariate kya hota hai?
Multi = Teen ya zyada | Variate = Variables — ek saath teen ya zyada variables ka analysis.

Goal: Complex interactions samajhna jo sirf ek ya do variables dekhne se nahi milte.
Real analogy: Student ke marks + attendance + study hours + part-time job — sab saath mein!
Multivariate kyun zaroori hai?
Univariate/Bivariate mein Simpson's Paradox hota hai — overall trend aur group-wise trend ulta ho sakta hai!
Example: Overall hospital mein mortality rate high hai. But jab severity of illness control karein toh acha hospital better perform karta hai!
Multivariate analysis = confounding variables dhundhna!
Multivariate Techniques
🔥
Heatmap (Correlation Matrix)
sns.heatmap(df.corr())
Saare numerical columns ke beech correlations ek hi view mein. Dark color = strong relation.
🏠 Housing: Area, Rooms, Floors, Price
Ek plot mein sab relations!
🔲
Pair Plot
sns.pairplot(df)
Har variable ko har doosre variable ke saath scatter plot. Diagonal pe distribution!
🌸 Iris dataset classic example
🏏 IPL batting stats comparison
🎨
Scatter (Color + Size Encoding)
hue=, size=
Scatter plot mein 3rd variable = color, 4th variable = dot size. 4 variables ek chart mein!
Area vs Price (color=City, size=Rooms)
🏠 Char variables ek chart mein!
📦

🏙️ Har city ke liye price distribution
📅 Har year ke liye sales trend

Heatmaps, Pairplots & Advanced Multivariate
Correlation Heatmap — Sab Relations Ek Nazar Mein
🐍 Python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.read_csv('mumbai_housing.csv')

# ─── Correlation Matrix ───
corr_matrix = df.select_dtypes(include='number').corr()
print(corr_matrix)

# ─── Heatmap ───
plt.figure(figsize=(10, 8))
mask = np.triu(np.ones_like(corr_matrix, dtype=bool))  # Upper triangle chhupa do

sns.heatmap(
    corr_matrix,
    annot=True,          # Numbers dikhao
    fmt='.2f',           # 2 decimal places
    cmap='coolwarm',     # Red = positive, Blue = negative
    mask=mask,           # Upper triangle hide
    vmin=-1, vmax=1,     # Scale fix
    linewidths=0.5,
    square=True,
    cbar_kws={'shrink': 0.8}
)
plt.title('Correlation Heatmap — Mumbai Housing Data', fontsize=14, fontweight='bold')
plt.tight_layout()
plt.show()

# ─── Top correlations dhundho ───
corr_pairs = corr_matrix.unstack().sort_values(ascending=False)
corr_pairs = corr_pairs[corr_pairs != 1.0]   # Self-correlation hata do
print("\nTop Positive Correlations:")
print(corr_pairs.head(5))
🔥 mask=np.triu() se upper triangle hide ho jaata hai — duplicate info nahi dikhti! coolwarm colormap: Red = positive, Blue = negative correlation.
Pair Plot — Har Variable vs Har Variable
🐍 Python
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.read_csv('ipl_batting_stats.csv')

# ─── Basic Pairplot ───
sns.pairplot(df[['runs', 'strike_rate', 'boundaries', 'innings']])
plt.suptitle('IPL Batting Stats — Pair Plot', y=1.02, fontsize=14)
plt.show()

# ─── Advanced: Color by category ───
sns.pairplot(
    df[['runs', 'strike_rate', 'boundaries', 'innings', 'team']],
    hue='team',              # Har team ka alag color
    diag_kind='kde',         # Diagonal pe KDE plot
    plot_kws={'alpha': 0.5}, # Transparency
    palette='Set1'
)
plt.suptitle('IPL Stats by Team — Pair Plot', y=1.02)
plt.show()

# ─── 4-variable scatter (color + size) ───
plt.figure(figsize=(10, 6))
scatter = plt.scatter(
    df['area_sqft'], df['price_lakhs'],
    c=df['bedrooms'],        # 3rd variable = color
    s=df['floor'] * 10,     # 4th variable = size
    alpha=0.6, cmap='viridis', edgecolors='white', linewidth=0.5
)
plt.colorbar(scatter, label='Bedrooms')
plt.xlabel('Area (sq ft)'); plt.ylabel('Price (₹ Lakhs)')
plt.title('4 Variables in One Chart!')
plt.show()
🌈 hue= parameter se ek extra dimension milti hai! Pairplot mein hue='team' → har team ka alag color → easily compare karo kaun outlier hai!

Pandas Profiling (ab ydata-profiling naam se) = ek line mein poora EDA report!

Complete Profiling Report — 3 Lines!
🐍 Python
import pandas as pd
from ydata_profiling import ProfileReport

# ─── Data Load ───
df = pd.read_csv('ipl_2023.csv')

# ─── Poora EDA report ek line mein! ───
profile = ProfileReport(
    df,
    title="IPL 2023 — Full EDA Report",
    explorative=True      # Extra deep analysis
)

# ─── HTML file mein save karo ───
profile.to_file("ipl_eda_report.html")
print("Report saved! Browser mein kholo.")

# ─── Jupyter Notebook mein directly dikhao ───
profile.to_notebook_iframe()

# ─── Minimal report (bade datasets ke liye faster) ───
profile_minimal = ProfileReport(df, minimal=True)
profile_minimal.to_file("quick_report.html")
⚡ Ek 10,000 row dataset ka full report = 30–60 seconds. Manual EDA = 2–3 hours. Time saved = Magical!

Advanced Options — Customize Your Report
🐍 Python
from ydata_profiling import ProfileReport
import pandas as pd

df = pd.read_csv('ecommerce.csv')

# ─── Custom Configuration ───
profile = ProfileReport(
    df,
    title="Ecommerce EDA — Detailed Report",

    # Missing values ka detailed analysis
    missing_diagrams={
        "bar": True,
        "matrix": True,
        "heatmap": True
    },

    # Duplicate rows check karo
    duplicates={"head": 10},

    # Correlation types specify karo
    correlations={
        "pearson": {"calculate": True},
        "spearman": {"calculate": True},
        "kendall": {"calculate": False},
        "cramers": {"calculate": True}   # Categorical ke liye!
    },

    # Large dataset ke liye sample lo
    samples={"head": 20, "tail": 20},

    # Report theme
    html={"style": {"primary_color": "#2563eb"}},

    # Dark mode!
    dark_mode=False
)

profile.to_file("custom_report.html")

# ─── Do datasets compare karo (drift detection!) ───
train_df = pd.read_csv('train.csv')
test_df  = pd.read_csv('test.csv')

train_report = ProfileReport(train_df, title="Train Data")
test_report  = ProfileReport(test_df,  title="Test Data")

# Side-by-side comparison!
comparison = train_report.compare(test_report)
comparison.to_file("train_vs_test.html")
🚀 compare() function = Data Drift Detection! Train vs Test data distribution mein fark dhundho — production ML mein bahut zaroori!
